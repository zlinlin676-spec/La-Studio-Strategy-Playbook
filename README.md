# 专栏定位

La Studio Strategy Playbook
不是知识科普栏目，
而是帮助经营者做出战略判断的工具库。

每篇文章必须回答：

1. 这个模型解决什么经营问题？
2. 大多数人在哪里用错？
3. La Studio有什么不同观点？
4. 如何落到真实生意？
5. 读者明天可以做什么？

# 写作原则

- 不堆砌正确但无用的废话
- 不把模型当答案
- 每个观点必须有商业依据
- 案例必须服务于判断
- 数据必须注明来源和年份
- 不写任何品牌都能套用的话
- 要有La Studio自己的结论
- 不要写一篇正确的文章
- 要写一篇别人只能从这里读到的文章

# 文章结构

## 开头
从商业痛点开头，要够痛，让经营者和创业者想要打开
需要商业思考的精准描述

## 01 一张图讲透一个商业模型
100-200个字+模型图片
介绍一个模型，要精简拆解

## 02 这个模型必须回答四个问题，比如
- 服务谁
- 解决什么
- 凭什么赢
- 放弃什么

## 03 案例
某品牌如何通过这个模型形成优势。

## 04 La Studio模型
定位四问模型。

## 结尾
什么都想要的品牌，
通常什么都得不到。
